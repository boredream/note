# JavaScript

## 函数

没有参数命名，动态使用的机制。

```python
# python
method(arg1: xxx, arg2: xxx)

xxx.method(arg2: xxx)
```



但因为是动态语言，直接传入一个对象，里面包含所有可选参

```js
// js
function method(args) {
 	console.log(args.arg1); 
}

method({"arg1": "value"});
```



## 技巧

### 非空判断

```java
// java
String s1 = null;
String s2 = s1 != null ? s1 : "";
```

```js
// js
var s1 = null;
var s2 = s1 || ""; // 对象自带一个 !=null 的布尔返回
```
